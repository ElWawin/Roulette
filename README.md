# 🎡 Interactive Roulette

A fully interactive, customizable roulette wheel game built with vanilla HTML, CSS, and JavaScript.

## Features

✨ **Interactive Wheel**
- Smooth spinning animation with realistic deceleration
- Canvas-based rendering for smooth performance
- Golden pointer indicator

🎨 **Full Customization**
- Add unlimited custom items
- Set custom colors for each item
- Adjust probability weights
- Delete items individually
- Reset to defaults or clear all

🔊 **Sound Effects**
- Spinning sound effect
- Stop/landing sound
- Click sounds for interactions
- Toggle sound on/off

✨ **Beautiful Animations**
- Smooth wheel rotation
- Confetti particle effects on win
- Button hover animations
- Panel transitions
- Result display animations

💾 **Data Persistence**
- Saves items to localStorage
- Custom items survive page refresh

📱 **Responsive Design**
- Works on desktop and mobile
- Adaptive layout for all screen sizes

## How to Use

1. **Open the page** - Load `index.html` in your browser
2. **Customize** - Use the control panel to add items, set colors and weights
3. **Spin** - Click "SPIN THE WHEEL!" or press Space bar
4. **Win** - See the result and celebrate with confetti!

## Files

- `index.html` - Main HTML structure
- `styles.css` - Complete styling and animations
- `script.js` - Game logic and wheel rendering

## Customization

### Add Items
1. Enter item name (max 15 characters)
2. Select color
3. Set probability weight (higher = more likely)
4. Click "Add"

### Adjust Spin Speed
Use the speed slider (5-30) to control how fast the wheel spins

### Sound Control
Toggle sound effects on/off with the switch
